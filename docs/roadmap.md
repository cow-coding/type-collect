# Roadmap

Tracks the areas we are actively shaping next for Tapistry. MVP work
is complete; everything below is reactive — ordered by the user signal
we are most likely to see first.

## Short term

- Continue polishing building sprites as edge-case proportions (tiny
  tiles, thumbnails, tall-sprite clipping) surface in real use.
- Keep onboarding copy and first-launch guidance aligned with the
  shipping village-builder loop — revisit after external feedback.
- Light economy tuning (coin drops, unlock prices) once we see how
  long players spend at each level.

## Medium term

- Seasonal / time-of-day sprite variants so the same 4×4 village keeps
  feeling fresh over weeks of play.
- Consider an optional prestige / soft level-cap break once players
  reach L20 so long-term typers have somewhere to go.
- Ambient layer (weather particles, birds, passing clouds) once the
  base catalog is settled.

## Not planned

- Paid IAP / premium skins. Tapistry stays free; tipping (Buy Me a
  Coffee / GitHub Sponsors) is the only funding path.
- Cloud sync / accounts. Village state stays fully local to preserve
  the privacy-first typing loop.
